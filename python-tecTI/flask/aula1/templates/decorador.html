Decorators em Python

Um decorator é uma função que recebe outra função como parâmetro e estende ou modifica seu comportamento sem alterar o código original da função decorada.

Por que utilizar?

  • Adicionar funcionalidades extras: Logs, cálculo de tempo de execução, verificação de permissões ou cache de resultados.
  • Reutilizar código: Evita a repetição (princípio DRY - Don't Repeat Yourself) aplicando a mesma lógica a diversas funções.
  • Metaprogramação: Permite alterar o comportamento de funções ou classes de forma limpa e expressiva.

Exemplo de Sintaxe

@meu_decorator
def minha_funcao():
    pass # A função original permanece intacta
